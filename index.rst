Soccer Object Msgs
==================

This is a collection of `ROS2 interface packages`_ for the soccer domain.
Custom message types specific to a soccer domain are defined in
the packages, and are explained in these docs.

The project is hosted on `Github`_.

See :ref:`soccer_object_msgs` page for more information.
